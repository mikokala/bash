# Salt-tila etätyöskentelyyn

## Käyttö

Aja skripti komennolla 'sudo sh high.sh'

Työasemaan saa nyt yhteyden sen IP-osoitteella VNC-viewerillä

## VNC ohjelmistot

[SSVNC](www.karlunge.com/x11vnc/ssvnc.html)

Saatavilla myös apt-jakeluna 'sudo apt-get install -y ssvnc'

[VNC Viewer](https://realvnc.com/en/connect/download/viewer/)

## Irssi

Käynnistetään komennolla 'irssi'
